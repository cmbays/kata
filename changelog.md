# Changelog

<Update label="Waves 0–K" description="Foundation through Dojo">

**Core engine, CLI, and Dojo training environment complete.**

Kata is a working development methodology engine with 2148 tests across 109 files. The system encodes Shape Up methodology as executable, composable stages with a self-improving knowledge system.

**Delivered across eleven waves:**

- **Waves 0–4** — Domain types (Zod schemas), services, persistence, intelligence layer, CLI polish
- **Wave 5** — Flavors, stage CRUD, execution adapters
- **Wave Orch** — 6-phase orchestration, StageVocabulary, MetaOrchestrator
- **Wave CLI** — Three-tier CLI (gyo → ryu → waza), `--json` everywhere, type refactors
- **Waves A–E** — Run state, JsonlStore, artifact/decision/approve commands, skill files, POC run, step/stage complete, gate set, rules, cooldown integration, init TUI, `kata watch`, `kata config`, lexicon + `--plain` mode
- **Wave K** — Dojo: diary entries, session generation, HTML design system, curated source registry

</Update>
